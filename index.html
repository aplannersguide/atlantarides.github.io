Atlanta Rides

Atlanta Rides is a volunteer group that meets twice a month at Code for Atlanta.

We are working with the Coord.co Shared Mobility API to display available vehicles from multiple operators on one map.

Our goal is to use technology to increase shared mobility to decrease Atlanta traffic.

The following Change.org petition aims to allow eBikes and eScooters on the Atlanta Beltline by implementing a new set of rules based on courtesy, sound and vehicle speed. Where space allows, Beltline planners could provide a separate path for bikes and scooters while completing the full rail loop.
